Work Experience

My sophomore year I interned as a Software Engineer at Alcatel-Lucent, a international telecoms company based in France. It was a great internship where I learned a lot about both low-level development as well as front end design. I programmed in C and Python, as well as Objective C and Java for mobile development. I had so much fun there that I decided to accept a return internship for next summer! Fun fact: The company has now been bought out by Nokia since I last worked there.

Currently I'm interning part-time as a software developer at State Farm here at the University's Research and Development Center. I'm working on an automated metadata collection tool written with the .NET framework. It crawls the State Farm databases and files in order to find SPI (sensitive personal information), and records when, where, and what type of information it found.

I also participate in HackIllinois, an annual hackathon here on campus. This is my 3rd year with the organization, and have enjoyed it more and more each time. This past year was the biggest one for the organization yet, with over 1,500 students from accross the country coming from schools like UC Berkeley, Purdue, and more!
